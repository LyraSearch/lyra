# Docusaurus orama Search Plugin

## Usage

Install the plugin:

```bash
npm install --save @orama/plugin-docusaurus
```

```bash
yarn add @orama/plugin-docusaurus
```

Add the plugin to your `docusaurus.config.js`:

```js
plugins: ['@orama/plugin-docusaurus']
```
