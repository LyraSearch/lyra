# Sort

To sort, Orama needs to know on which properties you want to sort.
To do that, the `sortSchema` has to be used during the creation.

```javascript copy
const db = await create({
  schema: {},
  sortSchema: {
    title: 'string',
    year: 'number',
    isTop: 'boolean',
    meta: {
      tag: 'string',
      rating: 'number',
      favorite: 'boolean',
    },
  }
})
const results = await search(db, {
  term: 'prestige',
  sort: {
    property: 'title', // or 'year', 'isTop'
  }
})
```

Orama upports sorting on 'string', 'number' and 'boolean'.

You can also specify nested properties using the '.' notation: 'meta.tag', 'meta.rating' and 'meta.favorite'

## Reverse order

Orama supports the reverse order specifying the key `order`:

```javascript

const db = await create({
  schema: {},
  sortSchema: {
    title: 'string',
    year: 'number',
    isTop: 'boolean'
    meta: {
      tag: 'string'
      rating: 'number',
      favorite: 'boolean',
    },
  }
})
const results = await search(db, {
  term: 'prestige',
  sort: {
    property: 'title', // or 'year', 'isTop'
    order: 'DESC' // 
  }
})
```
