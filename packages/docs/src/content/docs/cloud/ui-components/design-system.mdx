Orama Cloud offers a **ready-to-use** design system for your search and answer engine.

As of today, the design system features the following components:

- [**Search Button**](/cloud/ui-components/search-button): a button to trigger the search box modal.
- [**Search Box**](/cloud/ui-components/search-box): a full-featured searchbox with autocomplete and suggestions.
- [**Chat Box**](/cloud/ui-components/chat-box): a component to display the answer to a question.
- and much more...

## Storybook

Check out the [Design System](https://components.orama.com/) to see and try all the components available. 
We are constantly adding new components to the design system, so make sure to check it out regularly.

## Installation

To install the Orama UI Components library, you can use the following command, depending on your package manager and framework.

- [Web Components](#web-components)
- [React Components](#react-components)
- [Vue Components](#vue-components)
- [Angular Components](#angular-components)

If you are not sure, we recommend using the Web Components, as they are framework agnostic and can be used in any JavaScript environment.

##### Web Components

<Tabs>
  <TabItem label="npm" icon="seti:npm">
    ```bash

    ```
  </TabItem>
  <TabItem label="yarn" icon="seti:yarn">
