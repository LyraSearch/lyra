Orama Cloud offers a ready-to-use design system for your search and answer engine.

As of today, the design system features the following components:

- **Search Button**: a button to trigger the search box modal.
- **Search Box**: a full-featured searchbox with autocomplete and suggestions.
- **Search Results**: a list of search results with pagination and filters.
- **Facets**: a list of facets to filter search results.
- **Chat Box**: a component to display the answer to a question.
- and much more...

## Storybook

Check out the [Design System](https://components.orama.com/) to see and try all the components available. 
We are constantly adding new components to the design system, so make sure to check it out regularly.
