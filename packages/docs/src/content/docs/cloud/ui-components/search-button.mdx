## Installation

To install the <a href="https://components.orama.com/" target="_blank">Orama UI Components</a> library, you can follow the instructions [here](/cloud/ui-components/design-system#installation).

## Usage

The `Search Button` supports the following options.

| Option | Description | Default |
|:------ | ----------- | ------- |
| **colorScheme** | The selected color scheme. (`dark`,`light`,`system`) | `dark` |
| **size** | The size of the button. (`small`,`medium`,`large`) | `medium` |
| **label** | The label of the button. | `Search...` |

## Example

Here is an example of how to use the `Search Button` component in your application using **React**.

```jsx
import { OramaSearchButton } from '@orama/react-components';

<OramaSearchButton
  label="Search something..."
  size="large"
/>
```
