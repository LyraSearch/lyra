      <details open="true">
        <summary>What is an index?</summary>
        An [index](/cloud#base-concepts) is a structured representation of data that allows for efficient searching and retrieval of documents. Once an index is deployed, it remains immutable until the next deployment.
      </details>
      <details open="true">
        <summary>How can I add data to my index?</summary>
        You can create indexes through many different [datasources](/cloud/data-sources/introduction-to-data-sources). You can upload a [static file](/cloud/data-sources/custom-integrations/json-file), install one of the supported [native plugins](/cloud/data-sources/native-integrations/), or build your own custom integration via [REST APIs](/cloud/data-sources/custom-integrations/rest-apis).
      </details>
      <details>
        <summary>Does it integrate with Docusaurus?</summary>
        Yes! Simply install Orama's [Docusaurus plugin](/cloud/data-sources/native-integrations/docusaurus) and add unlimited search and answers to your documentation website in no time.
      </details>
      <details>
        <summary>Does it integrate with Shopify?</summary>
        Yes, you can [integrate your online shop](/cloud/data-sources/native-integrations/shopify) with Orama and leverage on the power of unlimited vector and hybrid search to make your products discoverable.

        Orama Cloud indexes are [immutable](/cloud#basic-concepts). To change the content of an index, you have to perform a re-deployment.
